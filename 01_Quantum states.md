A simple introductory project demonstrating fundamental quantum computing concepts using QuTiP (Quantum Toolbox in Python).

Overview:

This project covers:

1. Creation of basic qubit states:
2. Ground state |0⟩
3. Excited state |1⟩
4. Superposition state

Application of quantum gates using Pauli matrices:

1. X (bit-flip)
2. Y
3. Z
4. State transformation through gate operations
5. Visualization of qubit states on the Bloch Sphere
